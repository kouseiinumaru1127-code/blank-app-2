# アイドル診断（推し探し）アプリ

あなたの好みにぴったりのアイドルを見つけるための診断ツールです。
坂道グループ（=LOVE, ≠ME, ≓JOY）から、あなたに最適な「推し」を提案します。

## 🚀 アプリを試す
以下のURLから、ブラウザ上ですぐにアプリを試用できます。
**[https://blank-app-ljyg93y86a.streamlit.app/]**

## ✨ 主な機能
- **パーソナライズ診断**: 選択肢に回答することで、独自のアルゴリズムが最適なアイドルを抽出します。
- **データベース連携**: Supabaseを利用し、最新のアイドルデータに基づいた診断を行います。
- **直感的なUI**: Streamlitを採用し、誰でも簡単に操作できるインターフェースを提供しています。

## 🛠 使用技術
- **言語**: Python
- **フレームワーク**: Streamlit
- **データベース**: Supabase
